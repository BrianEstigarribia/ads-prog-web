Exercício CSS

Esse exercício é barbada. A estrutura do HTML e CSS já está montada. Você só precisa fazer as seguintes mudanças, utilizando CSS e sempre evitando duplicar regras:

  • Coloco o primeiro link do cabeçalho em negrito (font-weight: bold). Crie uma classe "menu-ativo" para esse estilo.
  • Garanta que somente os menus não ativos tenham o efeito do sublinhado (text-decoration: underline) quando se passa o mouse em cima
  • No link Ver mais da segunda tabela, defina que sempre que o mouse estiver em cima, fique em negrito.
  • Os nomes das disciplinas nas tabelas deve ficar em verde. Cria uma classe para aplicar na célula dessa coluna
  • Remova apenas a borda da tabela (border: none) utilizado para alinhar o formulário. As bordas das demais tabelas devem permanecer!
  • Transforme essa lista de items em uma lista ordenada e dê um espaçamento de 10px entre cada item da lista
  • Para cada seção (elemento section) defina uma borda superior de 1px preta e de traço continuo.
